Integrate Influx fitness business software's embeddable features into your WordPress install.

== Description ==

Influx is gym management software made simple. This plugin allows you to use the embeddable features without having to edit any HTML yourself.

== Installation ==

1. Upload the `influx` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your 'slug' from your Influx configuration on the Influx setup page in your WordPress install
4. Use the shortcode `[influx_planner]` in your posts or pages to embed your Influx Planner.
